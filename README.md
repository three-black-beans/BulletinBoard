# BulletinBoard
Mini Project for Java and Spring Study


## View
- Article List View
- Article Detail View
- Article Post View
- Article Patch View
- Comment Post View


## Function
- Article Post
- Article Patch
- Article Delete
- Comment Post
- Article Like
- Article View Count(Auto)
- Event Listener -> auto increment Article ID
- Pagination

- Java 11
- Spring 2.7.9
- Gradle-Groovy


## Dependencies 
- Spring web
- Thyme leaf
- Mongodb


## Used Tool
- Studio 3T
- IntelliJ
